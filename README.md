## Mahdi Maddahpour
 
Frontend developer in Stuttgart, Germany.
 
I work at [straiv](https://straiv.io) on hotel guest-journey software — online check-in,
self-service kiosks and mobile door opening, running in a few thousand hotels.
 
Most of my day is Angular and TypeScript: monorepo structure, state management, and the
unglamorous parts that decide whether a product actually holds up — form validation,
session handling, error reporting. Lately I've been working further down the stack as
well, on Python and FastAPI services running on AWS.
 
The part of this job I didn't expect to enjoy is that the software talks to physical
things: ID scanners, keycard encoders, door locks. Getting a browser to reliably drive
hardware in a hotel lobby, where the network is bad and the person at the kiosk is tired
and holding a suitcase, is a different problem from most frontend work.
